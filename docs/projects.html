Proyecto de prueba

Este es un proyecto de prueba para probar la conectividad entre mi sitio en GitHub Pages y una API sencilla desarrollada en Python y desplegada en un servidor externo, en este caso Python Anywhere.

Para este caso me basta con probar la conectividad y demostrar un mínimo de procesamiento por parte de la aplicación de Python, por lo que me limitaré a ejecutar una simple suma en dicha API y devolver el resultado.
